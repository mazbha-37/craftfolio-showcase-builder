## What is Vibe Coding?

This project was built using "vibe coding" — leveraging AI-powered development tools (specifically [Lovable](https://lovable.dev)) to transform natural language ideas into functional code. The process focused on rapid iteration, creative flow, and letting AI handle boilerplate while refining the user experience and design details.
